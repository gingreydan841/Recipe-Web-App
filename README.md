Welcome to my Recipe CookBook App!

This web application allows users to:

- View recipes
- Search recipes
- View recipe details
- Add new recipes with ingredients

Tools used:

- Node.js
- Express
- MySQL
- EJS
- HTML/CSS

How to Run:

1. Import the database
   SOURCE cookbook.sql;

2. Install dependencies
   npm install

3. Start the server
   node server.js

4. Open browser
   http://localhost:3000
